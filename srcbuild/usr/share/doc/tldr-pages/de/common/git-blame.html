git blame

Zeige den Commit-Hash und den letzten Autor jeder Zeile einer Datei. Weitere Informationen: https://git-scm.com/docs/git-blame.

  • Gib die Commit-Hashes und dem Autor jeder Zeile einer Datei aus:

git blame {{pfad/zu/datei}}

  • Gib die Commit-Hashes und dem Autor (inklusive E-Mail) jeder Zeile einer Datei aus:

git blame {{[-e|--show-email]}} {{pfad/zu/datei}}
