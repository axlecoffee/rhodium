choco feature

Interagiere mit Funktionen, die das Verhalten von Chocolatey verändern. Weitere Informationen: https://chocolatey.org/docs/commands-feature.

  • Gib eine Liste von verfügbaren Funktionen aus:

choco feature list

  • Aktiviere eine Funktion:

choco feature enable --name {{name}}

  • Deaktiviere eine Funktion:

choco feature disable --name {{name}}
