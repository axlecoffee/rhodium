hg

Mercurial - a distributed source control management system. Some subcommands such as commit have their own usage documentation. More information: https://www.mercurial-scm.org.

  • Execute a Mercurial command:

hg {{command}}

  • Display help:

hg help

  • Display help for a specific command:

hg help {{command}}

  • Check the Mercurial version:

hg --version
