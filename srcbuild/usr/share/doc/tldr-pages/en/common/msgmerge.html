msgmerge

Update an existing translation file from a new template. More information: https://www.gnu.org/software/gettext/manual/html_node/msgmerge-Invocation.html.

  • Update a translation file:

msgmerge {{[-U|--update]}} {{path/to/file.po}} {{path/to/messages.pot}}

  • Display help:

msgmerge {{[-h|--help]}}
