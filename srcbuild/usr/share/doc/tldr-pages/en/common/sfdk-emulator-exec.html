sfdk emulator exec

Execute a command on an emulator. More information: https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/40-testing-maintain/doc/command.emulator.adoc.

  • Execute a command on an emulator:

sfdk emulator exec {{emulator}} {{command}}

  • Run an interactive shell on an emulator:

sfdk emulator exec {{emulator}}
