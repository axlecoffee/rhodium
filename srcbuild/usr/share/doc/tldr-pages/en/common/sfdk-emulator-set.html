sfdk emulator set

Sets emulator properties. More information: https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/40-testing-maintain/doc/command.emulator.adoc.

  • Set a property:

sfdk emulator set {{name}} {{property}}={{value}}

  • Show emulator properties:

sfdk emulator show {{name}}
