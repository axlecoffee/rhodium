sfdk emulator start

Starts an emulator. More information: https://github.com/sailfishos/sailfish-qtcreator/blob/master/share/qtcreator/sfdk/modules/40-testing-maintain/doc/command.emulator.adoc.

  • Start an emulator:

sfdk emulator start {{name}}

  • Stop an emulator:

sfdk emulator stop {{name}}

  • Display emulator status:

sfdk emulator status {{name}}

  • Display the installed emulators:

sfdk emulator list
