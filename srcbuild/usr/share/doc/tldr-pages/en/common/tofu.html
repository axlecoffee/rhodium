tofu

Create and deploy infrastructure as code to cloud providers. Open-source fork of Terraform. More information: https://opentofu.org/.

  • Initialize a new or existing OpenTofu configuration:

tofu init

  • Verify that the configuration files are syntactically valid:

tofu validate

  • Format configuration according to OpenTofu language style conventions:

tofu fmt

  • Generate and show an execution plan:

tofu plan

  • Build or change infrastructure:

tofu apply

  • Destroy Tofu-managed infrastructure:

tofu destroy
