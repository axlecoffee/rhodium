dnf group

Manage virtual collections of packages on Fedora-based systems. More information: https://manned.org/dnf-group.

  • List DNF groups, showing installed and uninstalled status in a table:

dnf group list

  • Show DNF group info, including repository and optional packages:

dnf group info {{group_name}}

  • Install DNF group:

dnf group install {{group_name}}

  • Remove DNF group:

dnf group remove {{group_name}}

  • Upgrade DNF group:

dnf group upgrade {{group_name}}
