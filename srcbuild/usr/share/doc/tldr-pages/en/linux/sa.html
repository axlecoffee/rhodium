sa

Summarize accounting information about commands called by users, including basic information on CPU time spent processing and I/O rates. Part of the acct package. More information: https://manned.org/sa.8.

  • Display executable invocations per user (username not displayed):

sudo sa

  • Display executable invocations per user, showing responsible usernames:

sudo sa --print-users

  • List resources used recently per user:

sudo sa --user-summary
