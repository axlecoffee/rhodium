setfont

Set the terminal font for TTYs. More information: https://manned.org/setfont.

  • Change the terminal font:

setfont {{font}}.gz

  • Change the terminal font by specifying a path:

setfont /usr/share/kbd/{{mapping}}/{{font.ext}}

  • Double font size:

setfont {{[-d|--double]}}

  • Reset to the default font:

setfont
