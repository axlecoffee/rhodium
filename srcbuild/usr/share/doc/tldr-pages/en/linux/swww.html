swww

Efficient animated wallpaper daemon for Wayland. See also: swww-daemon. More information: https://github.com/LGFae/swww.

  • Set wallpaper:

swww img {{path/to/image}}

  • Set wallpaper to specified outputs:

swww img {{[-o|--outputs]}} {{output1,output2,...}} {{path/to/image}}

  • Restore last wallpaper:

swww restore

  • Kill daemon:

swww kill

  • Display output information:

swww query
