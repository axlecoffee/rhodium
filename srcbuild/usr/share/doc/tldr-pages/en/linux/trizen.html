trizen

Arch Linux utility for building packages from the Arch User Repository (AUR). More information: https://github.com/trizen/trizen/blob/master/TRIZEN.md.

  • Synchronize and update all AUR packages:

trizen -Syua

  • Install a new package:

trizen -S {{package}}

  • Remove a package and its dependencies:

trizen -Rs {{package}}

  • Search the package database for a keyword:

trizen -Ss {{keyword}}

  • Show information about a package:

trizen -Si {{package}}

  • List installed packages and versions:

trizen -Qe
