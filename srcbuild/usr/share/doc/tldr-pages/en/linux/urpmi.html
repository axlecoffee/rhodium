urpmi.update

Update the list of packages from a package repository in Mageia. Note: Mageia documentation uses medium and repository as synonymous. See also: urpmi, urpme, urpmi.addmedia, urpmi.removemedia, urpmf, urpmq. More information: https://man.linuxreviews.org/man8/urpmi.update.8.html.

  • Update all enabled media:

urpmi.update -a

  • Update specific media (including disabled media):

urpmi.update {{medium1 medium2 ...}}

  • Update all media that contain a specific keyword:

urpmi.update {{keyword}}

  • Update all configured media:

urpmi.update e
