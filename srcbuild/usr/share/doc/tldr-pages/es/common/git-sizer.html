git-sizer

Calcula varias métricas de tamaño de repositorios Git y te alerta de cualquiera que pueda causar problemas o inconvenientes. Más información: https://github.com/github/git-sizer.

  • Informa solo de las estadísticas que tienen un nivel de incidencia superior a 0:

git-sizer

  • Informa de todas las estadísticas:

git-sizer -v

  • Vea opciones adicionales:

git-sizer -h
