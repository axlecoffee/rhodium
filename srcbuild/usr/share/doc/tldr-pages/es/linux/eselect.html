eselect

Herramienta polivalente de configuración y gestión de Gentoo. Consta de varios módulos que se encargan de tareas administrativas individuales. Más información: https://wiki.gentoo.org/wiki/Eselect.

  • Muestra una lista de los módulos instalados:

eselect

  • Vea la documentación de un módulo específico:

tldr eselect {{módulo}}

  • Muestra un mensaje de ayuda para un módulo específico:

eselect {{módulo}} help
