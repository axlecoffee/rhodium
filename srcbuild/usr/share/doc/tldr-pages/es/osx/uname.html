uname

Imprime detalles sobre la máquina actual y el sistema operativo que se ejecuta en ella. Nota: para obtener información adicional sobre el sistema operativo, pruebe el comando sw_vers. Más información: https://keith.github.io/xcode-man-pages/uname.1.html.

  • Imprime el nombre del kernel:

uname

  • Muestra la arquitectura del sistema y la información del procesador:

uname -mp

  • Muestra el nombre, la versión y la versión del kernel:

uname -srv

  • Muestra el nombre de host del sistema:

uname -n

  • Muestra toda la información disponible del sistema:

uname -a
