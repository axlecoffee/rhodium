choco

El gestor de paquetes Chocolatey. Algunos subcomandos como install tienen su propia documentación de uso. Más información: https://chocolatey.org.

  • Ejecuta un comando Chocolatey:

choco {{comando}}

  • Llama a la ayuda general:

choco -?

  • Llama a la ayuda sobre un comando específico:

choco {{comando}} -?

  • Revisa la versión de Chocolatey:

choco --version
