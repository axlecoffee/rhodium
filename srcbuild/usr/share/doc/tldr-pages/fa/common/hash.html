hash

مشاهده مکان‌های ذخیره شده فایل‌های اجرایی در حافظه پنهان. اطلاعات بیشتر: https://www.gnu.org/software/bash/manual/bash.html#index-hash.

  • مشاهده مکان‌های دستورات ذخیره شده برای پوسته فعلی:

hash

  • پاک کردن جدول هش:

hash -r

  • حذف یک دستور خاص از جدول هش:

hash -d {{command}}

  • نمایش مسیر کامل command:

hash -t {{command}}
