aws history

Affiche l'historique de l'outil en ligne de commandes aws (l'enregistrement de l'historique de aws cli doit avoir été préalablement activé). Plus d'informations : https://docs.aws.amazon.com/cli/latest/reference/history/.

  • Liste l'historique des commandes avec leurs identifiants :

aws history list

  • Affiche les événements liés à une commande spécifiée par son identifiant :

aws history show {{identifiant_commande}}
