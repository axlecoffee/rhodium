git-sizer

Calcule diverses métriques de taille de dépôt Git et vous alerte de tout ce qui pourrait causer des problèmes ou des inconvénients. Plus d'informations : https://github.com/github/git-sizer.

  • Signaler uniquement les statistiques dont le niveau de préoccupation est supérieur à 0 :

git-sizer

  • Signaler toutes les statistiques :

git-sizer -v

  • Afficher les options additionnelles :

git-sizer -h
