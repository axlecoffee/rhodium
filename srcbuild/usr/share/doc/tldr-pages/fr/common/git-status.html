git status

Affiche les changements sur les fichiers dans la branche courante. Affiche les fichiers édités, déplacés, renommés, ajoutés, supprimés par rapport à la référence de la branche courante. Plus d'informations : https://git-scm.com/docs/git-status.

  • Affiche les fichiers modifiés qui n'ont pas encore été ajoutés pour le commit :

git status

  • Affiche les fichiers modifiés (version courte) :

git status {{[-s|--short]}}

  • Affiche les fichiers modifiés, sans tenir des comptes des fichiers non-suivis :

git status {{[-uno|--untracked-files=no]}}

  • Affiche les fichiers modifiés (version courte) avec les informations de branche :

git status {{[-sb|--short --branch]}}
