pwd

PowerShell में, यह कमांड Get-Location का उपनाम है। हालांकि, यह कमांड Command Prompt (cmd) पर उपलब्ध नहीं है। समान कार्यक्षमता के लिए इसके बजाय cd का उपयोग करें।

  • समकक्ष Command Prompt कमांड के लिए दस्तावेज़ देखें:

tldr cd

  • मूल PowerShell कमांड के लिए दस्तावेज़ देखें:

tldr get-location
