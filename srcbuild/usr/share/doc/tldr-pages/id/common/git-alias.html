git alias

Buat alias untuk perintah Git. Bagian dari git-extras. Informasi lebih lanjut: https://github.com/tj/git-extras/blob/master/Commands.md#git-alias.

  • Tampilkan daftar semua alias:

git alias

  • Buat alias baru:

git alias "{{nama}}" "{{perintah}}"

  • Cari alias yang sudah ada:

git alias ^{{nama}}
