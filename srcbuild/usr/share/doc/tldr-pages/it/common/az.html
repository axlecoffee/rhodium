az

Strumento ufficiale da linea di comando per Microsoft Azure. Alcuni comandi aggiuntivi, come login, hanno la propria documentazione. Maggiori informazioni: https://learn.microsoft.com/cli/azure/reference-index.

  • Effettua il log in ad Azure:

az login

  • Gestisci il tuo abbonamento ad Azure:

az account

  • Elenca tutti i dischi Azure Managed Disks:

az disk list

  • Elenca tutte le macchine virtuali di Azure:

az vm list

  • Gestisci i servizi Azure Kubernetes:

az aks

  • Gestisci le risorse della rete di Azure:

az network
