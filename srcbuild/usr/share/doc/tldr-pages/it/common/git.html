git

Sistema di controllo versione distribuito. Alcuni comandi aggiuntivi, come commit, hanno la propria documentazione. Maggiori informazioni: https://git-scm.com/docs/git.

  • Controlla la versione di Git:

git --version

  • Mostra informazioni di aiuto generali:

git --help

  • Mostra aiuto per uno specifico comando:

git help {{command}}

  • Esegui un comando Git:

git {{command}}
