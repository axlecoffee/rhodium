choco

Interfaccia a linea di comando per il gestore pacchetti Chocolatey. Alcuni sottocomandi come choco install hanno la loro documentazione. Maggiori informazioni: https://chocolatey.org.

  • Esegui comandi Chocolatey:

choco {{comando}}

  • Guarda il messaggio d'aiuto:

choco -?

  • Guarda il messaggio d'aiuto di un comando specifico:

choco {{command}} -?

  • Controlla la versione di Chocolatey:

choco --version
