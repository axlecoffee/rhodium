gem

Ruby 프로그래밍 언어용 패키지 관리자. 더 많은 정보: https://guides.rubygems.org.

  • 원격 gem을 검색하고 사용 가능한 모든 버전을 표시:

gem search {{정규_표현식}} --all

  • 최신 버전의 gem을 설치:

gem install {{젬_이름}}

  • 특정 버전의 gem을 설치:

gem install {{젬_이름}} --version {{1.0.0}}

  • 일치하는 최신 (SemVer) 버전의 gem을 설치:

gem install {{젬_이름}} --version '~> {{1.0}}'

  • gem 업데이트:

gem update {{젬_이름}}

  • 모든 로컬 gem을 나열:

gem list

  • gem 제거:

gem uninstall {{젬_이름}}

  • 특정 버전의 gem을 제거:

gem uninstall {{젬_이름}} --version {{1.0.0}}
