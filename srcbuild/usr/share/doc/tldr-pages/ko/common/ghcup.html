ghcup

Haskell 툴체인을 설치 도우미. Haskell 툴체인을 설치, 관리, 업데이트. 더 많은 정보: https://gitlab.haskell.org/haskell/ghcup-hs.

  • 대화형 TUI을 시작:

ghcup tui

  • 사용 가능한 GHC/cabal 버전 목록 나열:

ghcup list

  • 권장 GHC 버전 설치:

ghcup install ghc

  • 특정 GHC 버전 설치:

ghcup install ghc {{버전}}

  • 현재 “활성” GHC 버전을 설정:

ghcup set ghc {{버전}}

  • cabal을 설치:

ghcup install cabal

  • ghcup 자체를 업데이트:

ghcup upgrade
