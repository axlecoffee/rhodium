git authors

Git 저장소의 커밋 작성자 목록을 생성. git-extras의 일부. 더 많은 정보: https://github.com/tj/git-extras/blob/master/Commands.md#git-authors.

  • 커밋 작성자 목록을 AUTHORS 파일 대신 stdout에 출력:

git authors --list

  • 커밋 작성자 목록을 AUTHORS 파일에 추가하고 기본 편집기로 열기:

git authors

  • 이메일 주소를 제외한 커밋 작성자 목록을 AUTHORS 파일에 추가하고 기본 편집기로 열기:

git authors --no-email
