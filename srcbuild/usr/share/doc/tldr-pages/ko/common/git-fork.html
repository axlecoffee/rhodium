git fork

GitHub 저장소를 포크합니다. git clone과 유사하지만 먼저 포크를 수행합니다. git-extras의 일부입니다. 더 많은 정보: https://github.com/tj/git-extras/blob/master/Commands.md#git-fork.

  • URL을 사용하여 GitHub 저장소를 포크하고 클론:

git fork {{https://github.com/tldr-pages/tldr}}

  • 슬러그를 사용하여 GitHub 저장소를 포크하고 클론:

git fork {{tldr-pages/tldr}}
