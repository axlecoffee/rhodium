git whatchanged

최근 커밋 또는 파일의 변경 사항을 보여줍니다. 같이 보기: git log. 더 많은 정보: https://git-scm.com/docs/git-whatchanged.

  • 최근 커밋의 로그와 변경 사항 표시:

git whatchanged

  • 지정된 시간 범위 내에서 최근 커밋의 로그와 변경 사항 표시:

git whatchanged --since="{{2 hours ago}}"

  • 특정 파일 또는 디렉토리에 대한 최근 커밋의 로그와 변경 사항 표시:

git whatchanged {{경로/대상/파일_또는_폴더}}
