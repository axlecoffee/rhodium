npm team

npm 레지스트리에서 조직 내 팀을 관리. 더 많은 정보: https://docs.npmjs.com/cli/commands/npm-team.

  • 조직 내 팀에 사용자 추가:

npm team add {{조직:팀}} {{사용자_명}}

  • 팀에서 사용자 제거:

npm team rm {{조직:팀}} {{사용자_명}}

  • 조직 내 새 팀 생성:

npm team create {{조직:팀}}

  • 조직에서 팀 삭제:

npm team destroy {{조직:팀}}

  • 조직 내 모든 팀 나열:

npm team ls {{조직}}

  • 특정 팀의 모든 사용자 나열:

npm team ls {{조직:팀}}
