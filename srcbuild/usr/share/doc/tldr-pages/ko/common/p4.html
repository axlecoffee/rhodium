p4

Perforce 버전 관리 시스템. 더 많은 정보: https://www.perforce.com/manuals/cmdref.

  • Perforce 서비스에 로그인:

p4 login -a

  • 클라이언트 생성:

p4 client

  • 저장소에서 클라이언트 작업 공간으로 파일 복사:

p4 sync

  • 변경 목록 설명 생성 또는 편집:

p4 change

  • 파일을 편집을 위해 열기:

p4 edit -c {{변경_목록_번호}} {{경로/대상/파일}}

  • 새 파일을 열어 저장소에 추가:

p4 add

  • 변경 목록에 의해 수정된 파일 목록 표시:

p4 describe -c {{변경_목록_번호}}

  • 변경 목록을 저장소에 제출:

p4 submit -c {{변경_목록_번호}}
