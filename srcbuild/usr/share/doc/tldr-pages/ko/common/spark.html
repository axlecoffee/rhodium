spark

Laravel Spark 명령줄 도구. 더 많은 정보: https://spark.laravel.com.

  • API 토큰 등록:

spark register {{토큰}}

  • 현재 등록된 API 토큰 표시:

spark token

  • 새 Spark 프로젝트 생성:

spark new {{프로젝트_이름}}

  • Braintree 스텁과 함께 새 Spark 프로젝트 생성:

spark new {{프로젝트_이름}} --braintree

  • 팀 기반 결제 스텁과 함께 새 Spark 프로젝트 생성:

spark new {{프로젝트_이름}} --team-billing
