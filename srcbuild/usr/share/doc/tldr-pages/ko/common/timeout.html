timeout

명령을 일정 시간 제한 내에서 실행. 더 많은 정보: https://www.gnu.org/software/coreutils/manual/html_node/timeout-invocation.html.

  • sleep 10을 실행하고 3초 후 종료:

timeout 3s sleep 10

  • 시간 제한이 만료되면 명령에 [s]ignal을 전송 (TERM이 기본, 모든 신호 목록은 kill -l):

timeout --signal {{INT|HUP|KILL|...}} {{5s}} {{sleep 10}}

  • 시간 초과 시 전송된 신호를 stderr에 [v]erbose 출력으로 표시:

timeout --verbose {{0.5s|1m|1h|1d|...}} {{명령}}

  • 시간 초과 여부와 관계없이 명령의 종료 상태를 유지:

timeout --preserve-status {{1s|1m|1h|1d|...}} {{명령}}

  • 초기 신호를 무시할 경우 특정 시간 후 강제 KILL 신호 전송:

timeout --kill-after={{5m}} {{30s}} {{명령}}
