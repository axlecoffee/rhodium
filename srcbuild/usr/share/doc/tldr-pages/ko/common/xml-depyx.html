xml depyx

PYX (ESIS - ISO 8879) 문서를 XML 형식으로 변환. 더 많은 정보: https://xmlstar.sourceforge.net/docs.php.

  • PYX (ESIS - ISO 8879) 문서를 XML 형식으로 변환:

xml depyx {{경로/대상/입력.pyx|URI}} > {{경로/대상/출력.xml}}

  • stdin에서 PYX 문서를 XML 형식으로 변환:

cat {{경로/대상/입력.pyx}} | xml depyx > {{경로/대상/출력.xml}}

  • 도움말 표시:

xml depyx --help
