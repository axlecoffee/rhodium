archlinux-java

설치된 자바 환경 간 전환. 더 많은 정보: https://wiki.archlinux.org/title/Java#Switching_between_JVM.

  • 설치된 자바 환경 목록 나열:

archlinux-java status

  • 현재 기본 자바 환경의 짧은 이름 반환:

archlinux-java get

  • 기본 자바 환경 설정:

archlinux-java set {{자바_환경}}

  • 기본 자바 환경 설정 해제:

archlinux-java unset

  • 잘못되었거나 손상된 기본 자바 환경 구성 수정:

archlinux-java fix
