toolbox

Linux에서 컨테이너화된 명령줄 환경을 관리합니다. create와 같은 일부 하위 명령에는 자체 사용 설명서가 있습니다. 더 많은 정보: https://manned.org/toolbox.1.

  • toolbox 하위 명령 실행:

toolbox {{하위_명령}}

  • 특정 하위 명령(예: create, enter, rm, rmi 등)에 대한 도움말 표시:

toolbox help {{하위_명령}}

  • 도움말 표시:

toolbox --help

  • 버전 정보 표시:

toolbox --version
