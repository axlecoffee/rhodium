crane version

Print de versie van een binary. De versiestring is volledig afhankelijk van hoe de binary is gebouwd, dus je moet niet afhankelijk zijn van het versieformaat. Het kan zonder voorafgaande kennisgeving veranderen. Meer informatie: https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_version.md.

  • Toon versie:

crane version

  • Toon de help:

crane version {{[-h|--help]}}
