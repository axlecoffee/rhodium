docker rm

Verwijder een of meer containers. Meer informatie: https://docs.docker.com/reference/cli/docker/container/rm/.

  • Verwijder containers:

docker rm {{container1 container2 ...}}

  • Verwijdeer een container geforceerd:

docker rm {{[-f|--force]}} {{container1 container2 ...}}

  • Verwijder een container en de volumes:

docker rm {{[-v|--volumes]}} {{container}}

  • Toon de help:

docker rm {{[-h|--help]}}
