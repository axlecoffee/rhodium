xbps

Het X Binary Package System is het pakketbeheer die wordt gebruikt door Void Linux. For equivalent commands in other package managers, see https://wiki.archlinux.org/title/Pacman/Rosetta. Meer informatie: https://docs.voidlinux.org/xbps/index.html.

  • Bekijk de documentatie voor installeren en bijwerken van pakketten:

tldr xbps-install

  • Bekijk de documentatie voor verwijderen van pakketten:

tldr xbps-remove

  • Bekijk de documentatie om op zoek te gaan naar pakket- en repository-informatie:

tldr xbps-query
