chdir

Dit commando is een alias van cd in Command Prompt en vervolgens Set-Location in PowerShell. Meer informatie: https://learn.microsoft.com/windows-server/administration/windows-commands/chdir.

  • Bekijk de documentatie van het originele Command Prompt commando:

tldr cd

  • Bekijk de documentatie van het originele PowerShell commando:

tldr set-location
