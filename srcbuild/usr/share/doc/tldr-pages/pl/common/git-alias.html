git alias

Twórz skróty dla komend Gita. Jest częścią git-extras. Więcej informacji: https://github.com/tj/git-extras/blob/master/Commands.md#git-alias.

  • Wyświetl wszystkie aliasy:

git alias

  • Utwórz nowy alias:

git alias "{{nazwa}}" "{{komenda}}"

  • Wyszukaj istniejący alias:

git alias ^{{nazwa}}
