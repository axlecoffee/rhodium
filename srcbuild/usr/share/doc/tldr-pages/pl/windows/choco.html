choco

Interfejs wiersza polecenia dla menadżera pakietów Chocolatey. Niektóre podkomendy takie jak install posiadają osobną dokumentację. Więcej informacji: https://chocolatey.org.

  • Wykonanie podkomendy:

choco {{komenda}}

  • Wyświetlenie ogólnej pomocy:

choco -?

  • Wyświetlenie pomocy dla konkretnej podkomendy:

choco {{komenda}} -?

  • Wyświetlenie zainstalowanej wersji Chocolatey:

choco --version
