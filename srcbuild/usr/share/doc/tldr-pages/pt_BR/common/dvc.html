dvc

Controle de Versão de Dados: como o git mas para dados. Alguns subcomandos tal como commit tem suas próprias documentações de uso. Mais informações: https://dvc.org/.

  • Executa um subcomando do DVC:

dvc {{subcomando}}

  • Exibe a ajuda geral:

dvc --help

  • Exibe ajuda para um subcomando específico:

dvc {{subcomando}} --help

  • Checa a versão do DVC:

dvc --version
