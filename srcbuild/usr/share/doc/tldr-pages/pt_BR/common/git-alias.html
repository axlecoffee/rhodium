git alias

Cria comandos curtos para comandos Git. Parte do git-extras. Mais informações: https://github.com/tj/git-extras/blob/master/Commands.md#git-alias.

  • Lista todos os comandos curtos:

git alias

  • Cria um novo comando curto com o nome:

git alias "{{nome}}" "{{comando}}"

  • Procura por um comando curto existente:

git alias ^{{nome}}
