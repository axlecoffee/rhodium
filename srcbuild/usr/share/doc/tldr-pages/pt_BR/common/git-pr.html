git pr

Traz o código via checkout dos pull requests do GitHub localmente. Parte do git-extras. Mais informações: https://github.com/tj/git-extras/blob/master/Commands.md#git-pr.

  • Traz o código específico de um pull request:

git pr {{número_do_pr}}

  • Traz o código de um pull request para um remoto específico:

git pr {{número_do_pr}} {{remoto}}

  • Traz o código de um pull request da sua URL:

git pr {{url}}

  • Limpa pull requests antigos:

git pr clean
