git rm

Remove arquivos do índice do repositório e do arquivo local do sistema. Mais informações: https://git-scm.com/docs/git-rm.

  • Remove arquivo do índice do repositório e do arquivo local do sistema:

git rm {{arquivo}}

  • Remove um diretório:

git rm -r {{diretório}}

  • Remove arquivo do índice do repositório mas mantém o mesmo localmente:

git rm --cached {{arquivo}}
