distrobox-host-exec

Executar um comando no sistema host a partir do interior de um contêiner Distrobox. Subcomando de distrobox. Veja também: tldr distrobox. Mais informações: https://distrobox.it/usage/distrobox-host-exec.

  • Executa um comando no sistema host a partir do interior do contêiner Distrobox:

distrobox-host-exec "{{comando}}"

  • Executa o comando ls no sistema host a partir do interior do contêiner:

distrobox-host-exec ls
