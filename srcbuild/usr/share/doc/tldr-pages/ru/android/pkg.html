pkg

Утилита управления пакетами для Termux. Больше информации: https://wiki.termux.com/wiki/Package_Management.

  • Обновить все установленные пакеты:

pkg upgrade

  • Установить пакет:

pkg install {{пакет}}

  • Удалить пакет:

pkg uninstall {{пакет}}

  • Переустановить пакет:

pkg reinstall {{пакет}}

  • Поиск пакета:

pkg search {{пакет}}
