print

Вывод на печать текстового файла. Больше информации: https://learn.microsoft.com/windows-server/administration/windows-commands/print.

  • Печать текстового файла используя локальный принтер:

print {{путь\до\папки}}

  • Печать текстового файла используя сетевой принтер:

print /d:{{принтер}} {{путь\до\папки}}
