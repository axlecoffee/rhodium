git annotate

Her satırdaki dosyanın yanında en son commit değeri ve yazarını göster. Ayrıca git annotate yerine tercih edilen git blame sayfasına bakılması önerilir. git annotate, git dışındaki sürüm kontrol sistemlerine aşina olanlar için sağlanmıştır. Daha fazla bilgi için: https://git-scm.com/docs/git-annotate.

  • Bir dosyayı, her satırında son commit değeri ve yazarı bulunacak şekilde göster:

git annotate {{örnek/dosya}}

  • Bir dosyayı, her satırında son commit değeri ve yazarının e-postası bulunacak şekilde göster:

git annotate {{[-e|--show-email]}} {{örnek/dosya}}
