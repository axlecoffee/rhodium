git gc

Gereksiz dosyaları silerek yerel depoyu optimize et. Daha fazla bilgi için: https://git-scm.com/docs/git-gc.

  • Depoyu optimize et:

git gc

  • Agresifçe optimiize et (daha uzun sürer):

git gc --aggressive

  • Gevşek objeleri kesme (varsayılan olarak keser):

git gc --no-prune

  • Tüm çıktıları sessize al:

git gc --quiet

  • Tam kullanım için yardım göster:

git gc --help
