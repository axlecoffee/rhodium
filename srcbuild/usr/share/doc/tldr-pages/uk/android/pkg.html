pkg

Утиліта менеджеру пакетів для Termux. Більше інформації: https://wiki.termux.com/wiki/Package_Management.

  • Оновити всі встановлені пакети:

pkg upgrade

  • Встановити пакет:

pkg install {{пакет}}

  • Видалити пакет:

pkg uninstall {{пакет}}

  • Перевстановити пакет:

pkg reinstall {{пакет}}

  • Пошук пакету:

pkg search {{пакет}}
