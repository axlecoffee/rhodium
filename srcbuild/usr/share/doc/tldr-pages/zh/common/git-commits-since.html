git commits-since

显示从指定时间点以来的提交记录。 属于 git-extras 一部分。 更多信息：https://github.com/tj/git-extras/blob/master/Commands.md#git-commits-since.

  • 显示从昨天以来的所有提交：

git commits-since {{yesterday}}

  • 显示从上周以来的所有提交：

git commits-since {{last week}}

  • 显示从上个月以来的所有提交：

git commits-since {{last month}}

  • 显示从昨天下午2点以来的所有提交：

git commits-since {{yesterday 2pm}}
