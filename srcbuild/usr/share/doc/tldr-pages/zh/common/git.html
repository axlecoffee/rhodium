git

分布式版本控制系统。 类似如 commit、add、branch、checkout、push 等子命令都有自己的使用文档，可以通过 tldr git {{子命令}} 的形式查阅。 更多信息：https://git-scm.com/docs/git.

  • 执行 git 的子命令：

git {{子命令}}

  • 在自定义的 git 仓库根路径下执行子命令：

git -C {{仓库地址}} {{子命令}}

  • 在给定参数条件下，执行 git 的子命令：

git -c '{{配置项}}={{值}}' {{子命令}}

  • 显示帮助文档：

git --help

  • 显示 git 子命令的详细帮助文档（如 clone, add, push, log 等子命令）：

git help {{子命令}}

  • 检查 git 的版本号：

git --version
