powershell

此命令可能会被误认为是 PowerShell 的跨平台版本（以前称为 PowerShell Core），后者使用 pwsh 而不是 powershell。 Windows 中原始的 powershell 命令仍然可用于使用传统的 Windows PowerShell 版本（5.1 及以下版本）。 更多信息：https://learn.microsoft.com/powershell/module/microsoft.powershell.core/about/about_pwsh.

  • 查看最新的跨平台版本 PowerShell（版本 6 及以上）的命令文档：

tldr pwsh

  • 查看传统 Windows PowerShell（版本 5.1 及以下）的命令文档：

tldr powershell -p windows
