uv python

管理 Python 版本和安装。 更多信息：https://docs.astral.sh/uv/reference/cli/#uv-python.

  • 列出所有可用的 Python 安装：

uv python list

  • 安装某个版本的 Python：

uv python install {{版本}}

  • 卸载某个版本的 Python：

uv python uninstall {{版本}}

  • 搜索某个版本的 Python 安装：

uv python find {{版本}}

  • 锁定当前项目使用特定版本的 Python：

uv python pin {{版本}}

  • 显示 uv Python 安装目录：

uv python dir
