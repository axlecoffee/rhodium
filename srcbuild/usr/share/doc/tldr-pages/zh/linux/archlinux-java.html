archlinux-java

提供 Java 环境设置功能的一个帮助脚本。 更多信息：https://wiki.archlinux.org/title/Java#Switching_between_JVM.

  • 列出已安装的 Java 环境：

archlinux-java status

  • 返回当前默认 Java 环境的简称：

archlinux-java get

  • 设置默认的 Java 环境：

archlinux-java set {{java环境}}

  • 取消默认的 Java 环境：

archlinux-java unset

  • 自动设置默认的 Java 环境：

archlinux-java fix
