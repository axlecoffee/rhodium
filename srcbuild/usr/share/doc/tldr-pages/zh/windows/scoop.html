scoop

Windows 的命令行安装程序。 更多信息：https://scoop.sh.

  • 安装一个包：

scoop install {{包名}}

  • 删除一个包：

scoop uninstall {{包名}}

  • 更新所有已安装的包：

scoop update --all

  • 列出所有已安装的包：

scoop list

  • 显示一个包的信息：

scoop info {{包名}}

  • 搜索一个包：

scoop search {{包名}}

  • 移除所有包的旧版本并清理下载缓存：

scoop cleanup --cache --all
