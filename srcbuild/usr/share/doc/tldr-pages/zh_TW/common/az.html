az

Microsoft Azure 的官方客戶端工具。 此命令也有關於其子命令的文件，例如：login. 更多資訊：https://learn.microsoft.com/cli/azure/reference-index.

  • 登入到 Azure：

az login

  • 管理 Azure 訂閱資訊：

az account

  • 列出所有 Azure 託管磁碟：

az disk list

  • 列出所有 Azure 虛擬機：

az vm list

  • 管理 Azure Kubernetes 服務：

az aks

  • 管理 Azure 網路資源：

az network

  • 以互動模式啟動：

az interactive

  • 查看指令的說明：

az --help
